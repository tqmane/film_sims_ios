# FilmSims for iOS — 胶片模拟器

**为照片注入胶片的灵魂。**
一款再现胶片类型和相机品牌色调的照片编辑应用。

---

## 特性

| 功能 | 详情 |
|---|---|
| 🎞 **胶片模拟** | 一键应用各种预设中的心仪色调 |
| 🎚 **效果强度** | 在 0〜100% 的范围内精细调整模拟的应用量 |
| 🌾 **胶片颗粒** | 实时控制噪点量和颗粒感 |
| 🔍 **双指缩放** | 使用双指直观地放大或缩小预览图像 |
| 🗂 **滤镜分类** | 按品牌和类型整理的预设库 |
| 💾 **高质量导出** | 保存时保留原始分辨率和 Exif 信息 |
| ⚡ **GPU 高速处理** | 基于 Core Image 的 GPU 加速，带来流畅的编辑体验 |

---

## 技术栈

- **语言:** Swift
- **UI:** SwiftUI
- **图像处理:** Core Image
- **平台:** iOS 16+
