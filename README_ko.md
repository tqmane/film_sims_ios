# FilmSims for iOS — 필름 시뮬레이터

**사진에 필름의 숨결을.**
필름 스톡과 카메라 브랜드의 색조를 재현하는 사진 편집 앱.

---

## 특징

| 기능 | 상세 |
|---|---|
| 🎞 **필름 시뮬레이션** | 다양한 프리셋에서 원하는 색조를 원탭으로 적용 |
| 🎚 **효과 강도** | 0~100% 범위에서 시뮬레이션 적용량을 세밀하게 조절 |
| 🌾 **필름 그레인** | 노이즈 양과 입자감을 실시간으로 제어 |
| 🔍 **핀치 줌** | 두 손가락으로 미리보기 이미지를 직관적으로 확대/축소 |
| 🗂 **필터 분류** | 브랜드 및 장르별로 정리된 프리셋 라이브러리 |
| 💾 **고화질 내보내기** | 원본 해상도와 Exif 정보를 유지한 채 저장 |
| ⚡ **GPU 고속 처리** | Core Image를 통한 GPU 가속으로 쾌적한 편집 경험 제공 |

---

## 기술 스택

- **언어:** Swift
- **UI:** SwiftUI
- **이미지 처리:** Core Image
- **플랫폼:** iOS 16+
